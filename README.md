# Typeface Noto Sans

This package provides the Noto Sans font including all nine weights:

- Black (900)
- ExtraBold (800)
- Bold (700)
- SemiBold (600)
- Medium (500)
- Regular (400)
- Light (300)
- ExtraLight (200)
- Thin (100)

## Install

```
npm install typeface-noto-sans-full
```
